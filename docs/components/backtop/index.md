# Backtop 返回顶部

返回页面顶部的操作按钮。

## 基础用法

使用`<pd-backtop></pd-backtop>`标签，点击按钮直接返回页面顶部。

<div class="backtop-container">
Scroll down to see the bottom-right button.
<pd-backtop></pd-backtop>
</div>

```html
<template>
  Scroll down to see the bottom-right button.
  <pd-backtop></pd-backtop>

```
